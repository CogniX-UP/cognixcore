cognixcore.models¶

This module defines pydantic models for the session, nodes, ports

While pydantic is not a requirement for this library, this optional module provides utilities to work with the serialized versions of the corresponding cognix core objects.

This model is a requirement if cognixcore is installed with the Rest API dependency.
